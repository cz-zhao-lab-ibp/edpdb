Trouble Shooting

The following commands (options) are useful for debugging.
ECHO and VERBOSE . MAXERR .

The first thing you may try is to set echo option to level 2 and verbose option to a high level (say 6). These will let you know how the command interpretor work and give you more output than normal.

If the program stop 'unreasonably' after excuting a macro, it might be because that the macro contains the REWIND or GOTO commands which increase the error counter invisibly. This is for preventing an infinitive loop but could become annoying sometime. Reset maxerr to a large number before running the macro.

If you find the program crashes when it opens a non-existing file, you may need to recompile the program on your own computer. An alternative, but dirty fix is to copy all the files (e.g. PDB files and EDP files) that you need to run the programs into your working directory.
