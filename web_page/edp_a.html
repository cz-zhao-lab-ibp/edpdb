About the Program

1) This 06A version of EdPDB is available FREE OF CHARGE TO ACADEMIC AND NON-COMMERCIAL USAGES under the condition that the program package is used as a whole. Direct contact to the author for a permission is required before any other usages may take action, including taking codes from this package.
2) The current version of EdPDB, v06a, runs on SGI unix, linux and Window PC computers.It is distributed as a tar.gz file, which was created using the following unix command.

                "tar -zcf edpdb_v999-sgi.tar.gz edpdb"

Click here to download a 1.2 MB tar.gz file of the SGI package (a beta version). After unzipping the file (using "tar -zxf edpdb_v999-sgi.tar.gz"), one can find a ReadMe file which describes the installation. Contact the author to report installation problems.

Click here to download a 0.8 MB tar.gz file of the Linux package (version 06a) (a beta version). See tips on installation.

Click here to download a 0.5 MB rar file of the MS Window package (version 06a).

Click here to download a 1 MB tar.gz file of the Mac OSX package (version 06a) (thanks Dr. Doug Juers for the porting).
